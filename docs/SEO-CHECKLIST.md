# SEO Checklist (Puurfit)

## Pagina basics
- [ ] Unieke `<title>` per pagina
- [ ] Unieke `<meta name="description">`
- [ ] 1x H1 per pagina, logische H2/H3 structuur
- [ ] Canonical klopt (incl. .html nu)

## Technical
- [ ] robots.txt aanwezig + sitemap.xml link
- [ ] sitemap.xml valide en up-to-date
- [ ] OG + Twitter meta per key page
- [ ] JSON-LD: SportsActivityLocation / LocalBusiness op home

## Content
- [ ] Lessenpagina: Yoga/Pilates/Reformer + FAQ
- [ ] Prijzenpagina: duidelijke pakketten + CTA
- [ ] Contact: NAP (naam/adres/telefoon) zodra bekend
